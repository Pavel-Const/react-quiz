# QUIZ построитель тестов на React

## Возможности 
 - Прохождение теста
 - Валидация формы авторизации
 
 ## Планы на будущее
 - Прохождение созданного теста
 - Создание своего собственного теста


## Ссылка на сайт
   https://pavel-const.github.io/react-quiz/

### используются технологии
    React, React.Routing, CSS, HTML, WebPack, GIT
    
### Инструкция по запуску локально
1. Установить npm-зависимости: npm i
2. Запустить проект на локальном сервере для отладки: npm run dev
3. Cобрать проект: npm run build
